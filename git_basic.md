# Git basic

- `git init`
  - 깃 레포 만들기
- `git add`
  - staging area에 추가
- `git commit -m`

- `status`
  - 마지막 변경사항이 commit으로 부터 존재하나

- `git restore --staged ()`
  - stage로 ()를 내린다

- `git log --oneline`
  - 변경사항을 한눈에 본다

- `git remote add origin https://github.com/habeonplz/git-prac.git`
  - local git 과 git hub 연결
  
- `git remote -v`
  - 연결 확인
- ` git push origin master`
  - git hub에 올리기
